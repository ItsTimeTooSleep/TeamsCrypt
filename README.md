3. 拖入下载的安装包
4. **完成！** TeamsCrypt 图标已出现在工具栏

1. 刷新 Microsoft Teams 页面
2. 看到右上角 **🔒 加密徽章**
3. **点击启用** → 你的下一条消息已受保护
4. （可选）创建团队共享密码

## ❓ **常见问题**

### 🔸 需要技术知识吗？
**不需要**。只需基本电脑操作能力，设计目标就是“安装即用”。

### 🔸 会影响Teams性能吗？
**几乎无感知**。加密/解密在本地瞬间完成，不增加网络负担。

### 🔸 支持移动端吗？
目前专注于 **Teams Web 客户端**（桌面浏览器），覆盖最主要办公场景。

### 🔸 数据存储在哪里？
**所有数据本地处理**：
- 加密密钥 → 保存在你的浏览器存储
- 密码库 → 本地加密存储
- 无云端传输，无服务器存储

### 🔸 需要团队所有人都安装吗？
**推荐但不强制**：
- 发送方加密 → 仅需发送方安装并启用
- 接收方解密 → 需安装扩展（可自动解密或手动输入密码）
